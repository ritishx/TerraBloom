# 🌿 TerraBloom — eCommerce Website

A modern, nature-inspired **eCommerce website** designed and developed for **TerraBloom**, a makeup and skincare company.  
Built with **HTML, CSS, and JavaScript**, this website focuses on elegant UI, smooth UX, and ultra-fast performance — loading in just **0.8 seconds ⚡**.

---

## 🌸 Overview
**TerraBloom** combines nature and beauty through a clean, minimal, and responsive web experience.  
The platform showcases beauty products, skincare essentials, and makeup items from multiple eco-conscious brands, all within a lightweight and SEO-optimized interface.

---

## ✨ Features

- 🌿 **Fully Responsive Design** – Works seamlessly across mobile, tablet, and desktop.  
- ⚡ **Super-Fast Load Time** – Optimized architecture ensures <1s load speed.  
- 🔍 **Smart Product Search & Filter** – Quickly find items by category or brand.  
- 🛍️ **Elegant Product Cards** – With pricing, reviews, and smooth animations.  
- 🔒 **Minimal & Secure Checkout Interface**  
- 🧠 **SEO & Performance Optimized Codebase**  
- 🎨 **Brand-Driven Aesthetic** – Aligns with TerraBloom’s nature-inspired identity.
